# Identities

You can associate your web2 account to your DEMOS using the DEMOS sdk.

The following platforms are supported:

1. [Twitter](/sdk/web2/identities/twitter)
2. [Github](/sdk/web2/identities/github)
3. [Discord](/sdk/web2/identities/discord)
4. [Telegram](/sdk/web2/identities/telegram)
